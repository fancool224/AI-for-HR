## 字段缺失时的降级处理

| 缺失字段 | 影响 | 降级策略 |
|---|---|---|
| 流失类别 | 无法做主动/被动分析 | 报告中标注"仅总流失" |
| 司龄相关 | 无法做司龄分段 | 跳过 A2/B1/C7-C10/D5-D7/G1/G4 |
| 三阶意见 | 无法做三阶验证 | 跳过文本分析 Step 2 |
| 直接上级 | 无法做管理者画像 | 跳过 E1-E4/G4 |
| 是否高潜 | 无法做高潜预警 | 跳过 D1/D5/D8 |

## 司龄分段（6段）

| 段位 | 含义 |
|---|---|
| <1月 | 闪退（入职体验断裂）|
| 1-3月 | 试用期适应 |
| 3-6月 | 转正前观察期 |
| 6-12月 | 转正初期 |
| 12-24月 | 职业发展关键期 |
| 24月+ | 稳定期 |
