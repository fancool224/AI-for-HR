# 干预策略框架（四阶段 + 治标治本）

Agent 在产出"真因→策略"环节时使用。

## 真因分析（A3 双层下钻）

每条核心问题做"问题点 → 真因"双层下钻：

```
问题：【员工流失·主动】司龄1-3月流失15人（占主动流失60%）
  ↓
问题点1（占比60%）：直营部新员工流失12人
  ↓ 5why
  真因1.1：新员工带教SOP未落地（占比0.5）
  真因1.2：店长无新员工面谈机制（占比0.3）
  真因1.3：薪酬倒挂老员工（占比0.2）
问题点2（占比40%）：物流配货新员工流失3人
  ↓ 5why
  真因2.1：师徒制缺失（占比1.0）
```

**真因约束**：
- 必须落在"部门可控"层（不能写"员工心态")
- 必须带占比量化
- 必须回引"三阶交叉验证"中的具体证据

## 四阶段干预

每条真因配 1-2 条策略，按时间窗分类：

| 阶段 | 时间窗 | 典型动作 | 治标/治本 |
|---|---|---|---|
| 🚨 立即 | 0-30天 | 高风险员工一对一谈话、紧急补员、暂停问题岗招聘 | 治标 |
| 🔧 短期 | 1-3月 | 优化入职带教SOP、调整排班机制、面谈覆盖率达100% | 治标偏治本 |
| 🏗️ 中期 | 3-6月 | 薪酬方案调整、晋升评估标准透明化、师徒制建立 | 治本 |
| 🌱 长期 | 6月+ | 组织氛围改善、雇主品牌建设、人才素质模型 | 治本 |

**硬约束**：每月报告至少 1 条"🏗️中期 或 🌱长期"治本策略，否则就是打补丁。
